Understanding Base64 Encoding

Base64 is an encoding scheme that converts binary data into text using 64 printable characters. It ensures compatibility with systems that handle only textual data.

Character set typically includes Aâ€“Z, aâ€“z, 0â€“9, plus + and /. The URL-safe version substitutes these characters with - and _, and removes the padding symbol =.

Example

Original text: "Web"
ASCII bytes: 87 (W), 101 (e), 98 (b)
Binary: 01010111 01100101 01100010
6-bit chunks: 010101 110110 010101 100010
Base64: V2Vi

Thus, Base64 encoding transforms 3 bytes of data into 4 encoded characters.
